# THE BRANCH PREDICTOR · μARCH

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **PSĒPHOS**.

THE BRANCH PREDICTOR — the trained guess that handles every if you write (grew from David's loop+if collapse: 'the if is a prediction → have idiots train AI to do it eventually'; answers his two Qs: is it prefetch? is it the if-handler I reach through a wrapper?). THE PIVOT: in 2001 the predictor became a LITERAL NEURON — the perceptron predictor (Jiménez & Lin, HPCA 2001): history bits as ±1 inputs, signed learned weights, predict by sign(w₀+Σwᵢxᵢ), train with the perceptron rule online at GHz; cost scales LINEARLY with history (vs exponential counter tables) so it sees long correlations. ⚑ LIVE in-browser perceptron that actually LEARNS a branch (verified in node: ~100% on separable patterns loop/alt/block, ~50% on RANDOM = the Shannon floor where no predictor beats a coin). FULL LINEAGE: static/BTFN → Smith 2-bit saturating counter (1981) → Yeh&Patt two-level adaptive (1991) → McFarling gshare (1993) → tournament/hybrid (Alpha 21264, 1996) → PERCEPTRON (2001) → Seznec TAGE (2006) → modern TAGE-SC-L + hashed-perceptron hybrids (Zen/Intel, ~99%). ⚑ THE TIE: y=Σwᵢxᵢ is a weighted sum = a GATHER — a branch predictor is a one-neuron attention head over time ([[the-gather]]); but a single perceptron is LINEAR so it can't learn XOR (linearly inseparable) — the exact reason TAGE/piecewise-linear hybrids exist. ANSWERS: (1) NOT prefetch — its sibling (branch=guess control/the if; prefetch=guess data/the address; both speculation, both went perceptron). (2) YES the if-handler-through-a-wrapper — your if → compiler cmp/jne → front-end auto-consults it → speculate → resolve → commit or flush+rewind ~15 cycles; architecturally invisible (correct-as-if-in-order, mispredicts erased), you can only nudge it (__builtin_expect/PGO). Two-layer honest: real silicon, real lineage, exact mechanism, toy history size. Connects [[the-gather]] + [[ttu1]] (the neuron grown up) + [[the-rhythm-section]] (the pipeline it feeds).
